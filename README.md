# 🤖 Asistente SQL con IA

**Proyecto Final Integrador - Módulo: Python para IA (Máster en Inteligencia Artificial)**

## 📖 Descripción del Proyecto (Qué hace y por qué)
**¿Qué hace?** Esta es una aplicación web impulsada por IA (Opción A de la rúbrica) diseñada para procesar código SQL. Cuenta con dos funcionalidades principales: 
1. **Refactorización de código:** Toma consultas SQL desordenadas y las formatea automáticamente basándose en una guía de estilo estricta de ingeniería de datos.
2. **Explicación de código:** Traduce consultas SQL complejas a lenguaje natural paso a paso.

**¿Por qué lo hicimos?** Escribir y revisar código SQL suele ser un proceso manual que consume mucho tiempo en los equipos de datos. Esta herramienta estandariza el código para evitar errores, facilita las revisiones (Code Reviews) cumpliendo el principio DRY, y sirve como herramienta de aprendizaje para perfiles junior que necesitan entender consultas complejas.

## 🛠️ Tecnologías Utilizadas
* **Interfaz Web:** Streamlit
* **Modelo de Inteligencia Artificial:** Google Gemini 2.5 Flash
* **Conexión:** API de `google-generativeai`
* **Lenguaje:** Python 3.10+
* **Despliegue (Deployment):** Streamlit Community Cloud
